struct atom

Datatype representing atoms
To get a value from an atom, use the format 'ATOM_NAME.ATTRIBUTE'. For instance, to get the atomic-mass of 'Hydrogen', use 'hydrogen.atomicmass'.
atom hydrogen = {"HYDROGEN", "H", 1, 1.008};
hydrogen.name -> "HYDROGEN"
hydrogen.symbol -> "H"
hydrogen.atomicnumber -> 1
hydrogen.atomicmass -> 1.008
